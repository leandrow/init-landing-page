### Development

**Git clone**

```sh
git clone https://github.com/leandrow/init-setup.git
```

**Install** dependencies with [pnpm](https://pnpm.io/)

```sh
pnpm install
```

**To run** and **build**

```sh
npm run dev
```

```sh
npm run build
```
